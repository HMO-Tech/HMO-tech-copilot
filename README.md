# 📐 hmo-tech-copilot

> An AI-powered computational design web panel and conversational co-pilot tailored for generating Rhino & Grasshopper parametric Python scripts.

---

## 🌐 Overview | معرفی پروژه

**hmo-tech-copilot** is an advanced AI control panel designed to streamline the workflow for architects and computational designers. By leveraging deep language models via a secure Vercel backend, it translates natural language prompts into production-ready Python scripts for **Rhinoceros 3D** and **Grasshopper**, while offering multi-language chat assistance.

این پروژه یک پنل تحت وب و دستیار هوشمند مبتنی بر **هوش مصنوعی** است که دستورات متنی ساده را به اسکریپت‌های پایتون کاربردی برای محیط **Rhino** و **Grasshopper** تبدیل می‌کند. این ابزار با یک بک‌آند امن روی ورسل توسعه یافته و قابلیت تحلیل الگوریتم‌ها و گفتگو به دو زبان فارسی و انگلیسی را داراست.

---

## ✨ Features | قابلیت‌ها

* 🤖 **AI Script Generation:** Instant python code generation tailored for RhinoCommon and Grasshopper APIs.
* 🗣️ **Bilingual AI Assistant:** Fully understands and replies to engineering queries in both English and Persian.
* 📦 **Smart Code Box:** Automatically separates scripts from text, offering instant **Copy** and **Download .py** features.
* 🎨 **Interactive 3D UI:** Features a dynamic, live-rendered 3D tech mesh logo powered by `Three.js` that reacts to mouse movement.
* 🔒 **Secure Cloud Backend:** Serverless architecture on Vercel to protect API keys from client-side exposure.

---

## 🚀 Quick Deployment | راه‌اندازی سریع

To deploy your own version of this co-pilot on Vercel:

1.  Fork this repository.
2.  Create a new project on **Vercel** and link it to your forked repo.
3.  Add the following environment variable in Vercel Settings:
    * `GROQ_API_KEY` (Your secret key from Groq Console)
4.  Deploy, open the `.vercel.app` link, and start generating generative geometry scripts!
